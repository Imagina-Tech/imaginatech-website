3. Fluxo do Usuario

3.1 Fluxo do Assistente Financeiro

1. Login no Painel
→
2. Vincular WhatsApp
→
3. Enviar Comando
→
4. Bot Processa
→
5. Dados Sincronizados

Passo 1: Acesso ao Painel Financeiro

O usuario acessa https://imaginatech.com.br/financas/ e faz login utilizando sua conta Google (autenticacao via Firebase).

Passo 2: Vinculacao do WhatsApp

Nas configuracoes do painel, o usuario insere seu numero de WhatsApp para vincular ao assistente Claytinho.

Passo 3: Interacao via WhatsApp

O usuario envia mensagens para o numero comercial com comandos como:

  • "gastei 50 no mercado" - Registra uma despesa
  • "recebi 1000 de salario" - Registra uma receita
  • "qual meu saldo?" - Consulta o saldo atual
  • "extrato" - Lista transacoes do mes

Passo 4: Processamento pelo Bot

O webhook recebe a mensagem, o sistema processa o comando utilizando IA (processamento de linguagem natural) e executa a acao correspondente.

Passo 5: Sincronizacao

Os dados sao armazenados no Firebase e sincronizados em tempo real com o painel web.

5. Funcionalidades do WhatsApp

5.1 Comandos do Assistente Claytinho

Comando Exemplo Acao
Registrar Gasto "gastei 50 no mercado" Cria transacao de saida com valor R$50 na categoria Alimentacao
Registrar Receita "recebi 1000 de salario" Cria transacao de entrada com valor R$1000
Consultar Saldo "saldo" ou "quanto tenho?" Retorna saldo atual do mes
Ver Extrato "extrato" ou "transacoes" Lista ultimas transacoes do mes
Resumo Mensal "resumo" Envia resumo com entradas, saidas e saldo
Ajuda "ajuda" ou "comandos" Lista comandos disponiveis

7. Seguranca e Conformidade

7.1 Medidas de Seguranca

  • Criptografia em Transito: Todas as comunicacoes usam HTTPS/TLS

  • Validacao de Webhook: Verificacao de assinatura do Meta
  • Firestore Rules: Controle de acesso granular aos dados
