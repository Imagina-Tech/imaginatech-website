Politica de Privacidade

Ultima atualizacao: 27 de Janeiro de 2026

1. Introducao

A ImaginaTech ("nos", "nosso" ou "empresa") esta comprometida em proteger a privacidade dos usuarios de nossos servicos. Esta Politica de Privacidade descreve como coletamos, usamos, armazenamos e protegemos suas informacoes pessoais quando voce utiliza nosso site, servicos de impressao 3D, e nosso assistente virtual via WhatsApp ("Claytinho").

Importante: Ao utilizar nossos servicos, voce concorda com a coleta e uso de informacoes de acordo com esta politica.

2. Informacoes que Coletamos

2.1 Informacoes Fornecidas por Voce

  • Dados de Cadastro: Nome, e-mail, telefone, endereco (incluindo CEP, cidade, estado)
  • Dados de Pedidos: Arquivos para impressao 3D, especificacoes do servico, preferencias de entrega
  • Dados Financeiros: Transacoes registradas via assistente WhatsApp (para usuarios do painel financeiro)
  • Comunicacoes: Mensagens enviadas via WhatsApp, e-mail ou formularios de contato

2.2 Informacoes Coletadas Automaticamente

  • Dados de Uso: Paginas visitadas, tempo de permanencia, acoes realizadas
  • Dados Tecnicos: Endereco IP, tipo de navegador, dispositivo, sistema operacional
  • Cookies: Utilizamos cookies essenciais para funcionamento do site e autenticacao

2.3 Dados do WhatsApp (Assistente Claytinho)

  • Numero de telefone WhatsApp
  • Mensagens enviadas ao assistente (para processamento de comandos)
  • Dados financeiros informados pelo usuario (gastos, receitas, categorias)

3. Como Usamos Suas Informacoes

Utilizamos as informacoes coletadas para:

  • Prestacao de Servicos: Processar pedidos de impressao 3D, gerenciar entregas
  • Assistente Virtual: Responder comandos, registrar transacoes financeiras, fornecer resumos
  • Comunicacao: Enviar atualizacoes de pedidos, notificacoes importantes, responder duvidas
  • Melhoria dos Servicos: Analisar uso para aprimorar funcionalidades e experiencia
  • Seguranca: Prevenir fraudes, proteger contas, garantir integridade dos sistemas

4. Compartilhamento de Dados

Nao vendemos suas informacoes pessoais. Compartilhamos dados apenas nas seguintes situacoes:

  • Provedores de Servico: Firebase (Google) para autenticacao e armazenamento, Meta/WhatsApp para comunicacao via assistente
  • Transportadoras: Dados de endereco para envio de pedidos (ex: Melhor Envio)
  • Obrigacoes Legais: Quando exigido por lei ou ordem judicial
  • Protecao de Direitos: Para proteger nossos direitos, privacidade, seguranca ou propriedade

5. Armazenamento e Seguranca

5.1 Onde Armazenamos

Seus dados sao armazenados em servidores seguros do Google Firebase, localizados nos Estados Unidos, em conformidade com padroes internacionais de seguranca.

5.2 Medidas de Seguranca

  • Criptografia de dados em transito (HTTPS/TLS)
  • Autenticacao segura via Google Sign-In
  • Regras de seguranca do Firestore para controle de acesso
  • Monitoramento continuo contra acessos nao autorizados
  • Backups regulares dos dados

5.3 Retencao de Dados

Mantemos seus dados pelo tempo necessario para fornecer os servicos ou conforme exigido por lei. Dados financeiros do painel sao mantidos enquanto sua conta estiver ativa.

6. Seus Direitos (LGPD)

Em conformidade com a Lei Geral de Protecao de Dados (LGPD - Lei 13.709/2018), voce tem direito a:

  • Acesso: Solicitar copia dos seus dados pessoais
  • Correcao: Corrigir dados incompletos ou desatualizados
  • Exclusao: Solicitar a exclusao dos seus dados (quando aplicavel)
  • Portabilidade: Receber seus dados em formato estruturado
  • Revogacao: Revogar consentimento a qualquer momento
  • Informacao: Saber com quem seus dados foram compartilhados

Para exercer seus direitos, entre em contato conosco pelos canais indicados abaixo.

7. Uso do WhatsApp Business API

Nosso assistente "Claytinho" utiliza a API oficial do WhatsApp Business (Meta). Ao interagir com nosso assistente:

  • Suas mensagens sao processadas para entender comandos e fornecer respostas
  • Dados financeiros informados sao armazenados em sua conta pessoal no Firebase
  • Nao armazenamos historico de conversas alem do necessario para processar comandos
  • Voce pode desvincular seu numero a qualquer momento nas configuracoes do painel

O uso do WhatsApp tambem esta sujeito a Politica de Privacidade do WhatsApp.

8. Cookies e Tecnologias Similares

Utilizamos cookies para:

  • Autenticacao: Manter voce logado em sua conta
  • Preferencias: Lembrar suas configuracoes
  • Analiticos: Google Analytics para entender uso do site (dados anonimizados)

Voce pode configurar seu navegador para recusar cookies, mas isso pode afetar a funcionalidade do site.

9. Menores de Idade

Nossos servicos nao sao direcionados a menores de 18 anos. Nao coletamos intencionalmente dados de menores. Se tomarmos conhecimento de que coletamos dados de um menor, tomaremos medidas para excluir essas informacoes.

10. Alteracoes nesta Politica

Podemos atualizar esta Politica de Privacidade periodicamente. Notificaremos sobre mudancas significativas por e-mail ou aviso em nosso site. A data de "ultima atualizacao" no topo indica quando a politica foi revisada pela ultima vez.
